# i18n CLI

## Usage

```bash
npm install @divlook/i18n-cli
npx divlook-i18n
```

## Develop guide

### Setup

```bash
nvm use # v16
npm install
```

### Develop

```bash
npm run test.watch
```

### Build

```bash
npm run build # output: ./dist/i18n-cli.cjs
```

### Execute

```bash
npm run exec
```
